# mysql-docker-starter
Starter for a mysql server in Express using a docker approach for a quick local development setup


**Uses:**
- Node v18.17.1 (use NVM 18)


## To install
``` JS Terminal
`npm install`
```

## To Run Server
``` JS Terminal
`cd server`
```
```
`npm run dev`
```

## To Close Server
```JS Terminal
ctrl + c
```
